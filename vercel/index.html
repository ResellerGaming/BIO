Environment Variables (ENV) di Vercel berfungsi untuk menyimpan data rahasia seperti API key, token, atau konfigurasi yang tidak boleh ditulis langsung di kode sumber. Contohnya: DATABASE_URL, API_TOKEN, atau SECRET_KEY.

4Ô∏è‚É£ Gunakan ENV di Kode

Setelah disimpan, kamu bisa mengakses variabel tersebut dalam kode:
